## Usage

### Basic Usage

```tsx
import { Toast } from '@lightmind/ui'

export function Example() {
  return <Toast />
}
```

### Advanced Example

```tsx

export function AdvancedExample() {

    <Toast
      className="custom-class"
```

## Usage Guidelines

### Do's ✅

- Use Toast for its intended purpose
- Apply consistent styling across your application
- Consider accessibility when implementing
- Test across different screen sizes and devices

### Don'ts ❌

- Avoid nesting too many Toast components
- Don't use for unintended purposes

### Best Practices

Always ensure that your Toast implementation follows accessibility guidelines and provides a good user experience across all devices.
